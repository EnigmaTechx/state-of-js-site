Dive Into the World of JavaScript

Since its inception in 1995, JavaScript or JS, for short, has been used by many developers for building wide range of web applications on the internet and manipulating behaviour of web browsers. As it evolved, several communities of programmers and engineers sought to use frameworks to incorporate specific features of JavaScript. A framework is basically a collection of libraries that provide developers with pre-defined code to use for routine programming features⇗. They make it easy to build websites and web applications without having to write code from scratch and allows developers to focus on other aspects tasks such as the User Interface and User Experience of websites (UI/UX). Currently, there are over 83 JavaScript frameworks and libraries (See more), with some developers joking that there are more frameworks than web applications that use them. Here, we intend to look at the most popular ones among developers and leading tech industries.

Factors to Consider When Adopting a JS Framework

Usability

Fresh off an HTML-CSS-JS bootcamp, will I be able to understand how to use this framework?

Speed

To avoid writing unnecessary code from scratch, how fast can I build a web app with this framework?

Community

When I get stuck, are there other developers experienced with this framework that can help me solve my problem?
